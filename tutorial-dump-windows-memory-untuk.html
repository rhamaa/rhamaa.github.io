Tutorial Dump Windows Memory Untuk Forensic

28 Jul 2016 on Digital forensic

Assalamualaikum…

</b> Dump Windows Memory - Proses analys barang bukti kejahatan haruslah ditangani dengan tepat agar barang bukti tersebut tidak hilang maupun rusak saat dilakukan forensic. Dalam kejahatan dunia maya atau Cyber Crime semua bukti atau Evidence yang ditemukan oleh Digital Forensic Investigator sangatlah penting untuk mengungkap motif dibalik kejahatan yang dilakukan tersangka. Dalam proses pengumpulan bukti ada saat nya kita harus mencari bukti tersebut pada komputer tersangka, sehingga dapat ditemukan bukti ataupun track kejahatan yang dilakukan, untuk itulah sebagai Digital Forensic Investigator kita harus bisa melakukan Dump Memory agar bisa kita analys lebih lanjut dikomputer kita sendiri tanpa harus lagi menyentuh komputer tersangka untuk menghindari adanya kerusakan barang bukti atau Evidence yang asli.

Dump Windows Memory Dengan DumpIT

              DumpIT adalah utility yang digunakan untuk melakukan generate terhadap physical memory pada system operasi windows baik itu 32 bit dan 64 bit. DumpIT sendiri cukup mudah digunakan, hanya dengan melakukan double click pada file DumpIt.exe proses dump bisa langsung dijalankan. agar proses dapat dilanjutkan DumpIt hanya mempunyai satu pertanyaan apakah ingin melanjutkan proses atau tidak serta hasil dari proses dump memory akan disaimpan dengan format RAW pada directory tempat DumpIt dijalankan.

Proses Dump Memory

Jalankan DumpIt.exe dengan melakukan Double Click. Pada pertanyaan “Are you sure want to continue ?” tekan tombol y untuk melanjut dan tombol n untuk keluar.

Dump Windows Memory Dengan FTK Imager

             Sama hal nya seperti DumpIt,   AcccessData FTK Imager dapat digunakan untuk memperoleh live memory dan paging file pada system 32 bit maupun 64 bit. Untuk penggunaan nya cukup mudah hanya dengn melakukan click pada icon Capture Memory pada FTK Imager.

Proses Dump Memory

Sebelum pastikan AccessData FTK Imager sudah diinstall ( link download dibawah ).

Jalankan AccessData FTK Imager.

Akan keluar Window baru, silahkan Browse Destination Path nya untuk menyimpan hasil dump lalu click Capture .

Proses Dumping akan dimulai, silahkan tunggu hingga selesai.

Proses Dumping sudah selesai Click Close.
( Result dari memory dump dengan size 2,080,374,784 bytes )
